Map Dvar For Server Admins 


Admins can use this Dvar to change ambient sound and fx.

To use this Dvar just add to your Game server .cfg file (example server.cfg)

set storm_nofx "0"			// Set to 1 to remove all rain fx and sounds,some fx and local sounds will remain (default=0)

----------------------------
MIP protection

The Map Integrity Program is a small code that checks your map iwd size(value) that is generated by the game engine and then compares it to the value the mapper has entered into the code.
If the values are different (iwi's or mp3 files changed) then the code will abort the map.
